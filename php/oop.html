OOP的學習筆記

Codeigniter3的一些學習隨筆

->

箭頭符號->引用(調用)物件的方法method或屬性property

$this

OOP允許物件使用$this來引用他自己, 在方法裡使用$this就好比在類別class外面的物件名稱($obj)一樣

public function login_user($username,$password){//使用者登入 $this->db->where('username', $username);//比對輸入username欄位 $result = $this->db->get('users'); $db_password = $result->row(7)->password;//資料庫第七個欄位為password if(password_verify($password, $db_password)){//如果輸入密碼與資料庫解密後密碼相同 return $result->row()->id;//該筆即為相對帳號資料，取出資料表中的id property }else{ return false; } }
